# Portfolio Website - Project Showcase

## What It Is

A modern full-stack portfolio website with a unique retro aesthetic. Built to showcase my skills, projects, and provide an easy way for visitors to get in touch. The entire site is managed through a custom admin dashboard - no coding required to update content!

---

## Tech Stack

**Frontend:**
- React (TypeScript)
- Custom CSS with retro design system
- Vite

**Backend:**
- Node.js & Express
- MongoDB & Mongoose
- JWT Authentication

**Design:**
- Figma (design & illustrations)
- Custom SVG artwork
- Google Fonts (Protest Strike, Spicy Rice, Squada One)
